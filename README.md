# SnatchIt Downloader
### A simple program for downloading videos and music from YouTube
#### How to use:
Grab the latest version from the Releases tab, download the executable and run.  
Run the executable, insert a youtube video link, select video or audio and press save.  
Then, select a download destination and download.

---

#### Building:
You'll need:
- Pyside6 installed
- yt-dlp, ffmpeg and ffprobe in the build dir
- nuitka compiler  

---

To compile, run:  
    `nuitka ./downloader.py`
